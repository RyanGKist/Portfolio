WiFi Finder

This is a project created using Angular-4 on the frontend, and postgreSQL on the backend. WiFi Finder allows the user to locate WiFi hotspots around them, or in specific areas. Based on data pulled from backend data entered from user. The project utilizes Angular's observables, as well as data response following JSON API's formatting conventions.
